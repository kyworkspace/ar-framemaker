# 🛠️ Motion Asset Master Tool

**Block Guardian** 및 2D 웹 게임 개발을 위한 브라우저 기반의 강력한 모션 에셋(스프라이트) 전처리 도구입니다. 

무거운 설치형 그래픽 툴이나 파이썬 환경 없이, 웹 브라우저 단에서 동영상 프레임 추출, 크로마키 배경 제거, 프레임 편집, 그리고 스프라이트 시트/GIF 생성을 한 번에 처리할 수 있습니다.

## 🎯 사용 용도
- 3D/AI 생성 비디오에서 2D 게임용 캐릭터 모션 프레임 추출
- 단색(크로마키) 배경을 가진 영상 및 이미지의 배경을 투명하게 제거 (알파 채널 적용)
- 프레임 단위의 미세 편집 (순서 변경, 불필요한 프레임 삭제, 복제)
- 웹 게임 엔진에 바로 올릴 수 있는 **Sprite Sheet(PNG)** 및 **투명 GIF** 제작

---

## ✨ 주요 기능 (Features)

* **🎥 동영상 프레임 추출 & 크로마키 엔진**
  * 업로드된 영상을 지정한 FPS 간격으로 쪼개어 이미지로 추출합니다.
  * Canvas API 기반의 고속 크로마키 알고리즘을 탑재하여, 추출과 동시에 지정한 배경색을 투명하게 날려버립니다.
  * 색상 픽커와 강도(Threshold) 슬라이더를 통해 배경 제거 범위를 세밀하게 조절할 수 있습니다.
* **📁 다중 이미지 병합 및 강제 PNG 변환**
  * 이미 추출해 둔 여러 장의 JPG/WebP/PNG 이미지를 한 번에 업로드하여 리스트에 누적시킬 수 있습니다.
  * 업로드된 모든 이미지는 내부적으로 투명도를 지원하는 순수 PNG 데이터로 자동 변환됩니다.
* **✂️ 강력한 프레임 편집 (D&D)**
  * **Drag & Drop:** 프레임을 마우스로 끌어당겨 애니메이션 재생 순서를 직관적으로 바꿀 수 있습니다.
  * **일괄 편집:** 전체 선택, 선택 삭제, 선택 복제(Copy), 선택 이미지 크로마키 적용 등 툴바를 통한 일괄 제어를 지원합니다.
  * **자동 넘버링:** 순서를 바꿀 때마다 `prefix_0000.png` 형태로 파일명을 자동 재정렬하는 옵션을 제공합니다.
* **🔍 상세 뷰 및 이름 변경 (Modal)**
  * 프레임을 더블 클릭하면 이미지를 크게 확대해서 확인하고, 해당 프레임의 파일명을 즉시 개별 수정할 수 있습니다.
* **📦 다중 포맷 Export 지원**
  * **Sprite Sheet:** 웹 게임 최적화를 위해 모든 프레임을 가로로 길게 이어 붙인 1장의 투명 PNG로 다운로드.
  * **GIF:** 지정한 속도(FPS)에 맞춘 투명 애니메이션 GIF 생성.
  * **ZIP:** 현재 리스트에 있는 모든 개별 프레임들을 하나의 ZIP 파일로 묶어서 다운로드.

## 🚀 사용 방법 (How to Use)

### 1. 실행 방법
별도의 백엔드 서버가 필요 없는 100% 클라이언트 사이드(바닐라 JS) 웹 애플리케이션입니다.
- **로컬 환경:** VS Code의 `Live Server` 확장을 사용하여 `index.html`을 실행하세요. (또는 터미널에서 `npx serve` 사용)
- **웹 호스팅:** GitHub Pages 등을 통해 호스팅하면 브라우저 주소창 접속만으로 즉시 사용할 수 있습니다.

### 2. 작업 흐름 (Workflow)
1. **업로드:** [1. 동영상에서 추출] 또는 [2. 이미지 프레임 추가]를 통해 소스를 올립니다.
2. **배경 제거:** 크로마키 설정 패널에서 제거할 배경색(예: 🟢 연두색)을 스포이트로 찍고, 제거 강도를 조절한 뒤 [크로마키 적용 추출]을 누릅니다.
3. **편집:** 하단 미리보기 영역에서 불필요한 눈 깜빡임, 어색한 동작 등을 체크하여 [선택 삭제] 하거나 순서를 드래그하여 맞춥니다.
4. **출력:** 원하는 결과물 형태(GIF, Sprite Sheet, ZIP) 버튼을 눌러 에셋을 다운로드하고 게임 엔진에 적용합니다!

## 🛠️ 기술 스택 (Tech Stack)
- **HTML5 / CSS3 / Vanilla JavaScript**
- **HTML5 Canvas API:** 픽셀 데이터 조작 및 크로마키 알고리즘 구현
- **[JSZip](https://stuk.github.io/jszip/):** 브라우저 단 ZIP 압축 처리
- **[gif.js](https://jnordberg.github.io/gif.js/):** 클라이언트 사이드 GIF 인코딩
